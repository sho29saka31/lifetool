
# 🛠️ Life-Tool
**「人生のすべてを、オープンソースな1画面に集約する。」**

`Life-Tool` は、500種類以上のユーティリティ、AIアシスタント、自己管理ツールを統合した、GitHub Pagesで動作する個人向けポータルサイトです。

---

## 🚀 プロジェクトのビジョン
現代のデジタルライフにおいて分散したツールを、一つの「静的サイト」に集約します。
- **サーバー**: GitHub Pagesによる永久無料ホスティング。
- **データ主権**: データはユーザー自身のもの。Google Drive APIを介して、個人のクラウドと同期します。
- **プライバシー**: サーバーを介さないため、第三者がデータに触れることは物理的に不可能です。

---

## 🛠️ 技術スタック & アーキテクチャ
保守性と拡張性を最大化するために「プラグイン方式（独立構成）」を採用しています。

- **フロントエンド**: HTML, CSS, JavaScript
- **ストレージ**: IndexedDB (メインDB), LocalStorage (設定用)
- **同期**: Google Drive API (App Data Folder)
- **認証**: Google OAuth 2.0
- **展開**: GitHub Pages

## 🔐 セキュリティ & プライバシー
1. APIキーの取り扱い
Google AI Studio (Gemini API) 等の個人キーは、LocalStorageにのみ保存され、ソースコードやGitHubのサーバーに保存されることはありません。

2. データ同期
Googleログインを通じて、ユーザー自身のGoogle Drive（アプリ専用隠しフォルダ）にデータを保存します。開発者がユーザーのデータを見ることはできません。
